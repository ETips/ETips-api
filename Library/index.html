检索图书

GET /library/search/<string:keyword>/page/<int:page>

Parameters

name description
keyword 关键字
page 页码

Response

    {
        "press_time": "2015.1", # 出版时间
        "num": "1", # 序号
        "index_num": "TP312JA/H97ah",  # 索取号

        "author": "(美)Cay S. Horstmann著；张若飞译",# 作者

        "total": "3", # 馆藏
        "ctrlno": "572952", # 图书馆系统控制号(在图书馆的唯一编号)
        "left": "2"  # 剩余
    },
    .......
]
