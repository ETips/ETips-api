WYU OPEN API

五邑大学校园开放接口(非官方版)

缘由

ETips因为校园网不时抽风，有太多的不稳定，所以用户体验比较差。所以隐痛写下这个开放接口。 接口开发是有限的，因为部分校园接口需要在内网环境下使用，然后这个校园开放接口是挂在外网，所以无法实现。

当然啦，写这个不只是为了ETips服务，而是我打算用这个来交web开发的课程设计，所以这个项目在本学期开发激情还是杠杠的。

开发

本网站采用Python开发，原因很简单——爬虫比较简单。

如果你有啥需求、发现bug，请在issue中提交反馈。 需求不一定接受，但bug肯定会及时修复。
